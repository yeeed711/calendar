3. 새로 알게된 점

- 일요일의 열을 콜그룹으로 묶어 폰트색상을 변경하려고 했는데 콜그룹에는 적용가능한 속성이 정해져있었다.
- CSS selector 를 이용해서 `td:nth-child(1)`, `th:nth-child(1)` 를 사용해 색상을 변경했다. 모든 열의 첫번째 자식이므로 `nth-child(n)` 선택자를 더 잘 다룰 수 있어야 할 것 같다.
